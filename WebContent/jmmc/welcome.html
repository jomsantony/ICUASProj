ICUAS-JMMC


Welcome to Intensive Care Unit Automation Project(ICUAS-JMMC). This project aims to automate ICU related workflows for Hospitals possibly by making use of Tab/Smart phone working in web/android plaform. This Module focusses on providing easy access to existing ICU data in an ubiquitous manner. To start with Click on the Bed Number to view the basic details of the occupied patient. This can be followed by clicking the different Facets like:

Patient Data
Charts
Lab Results
Clinical Orders
Progress Notes
Images
